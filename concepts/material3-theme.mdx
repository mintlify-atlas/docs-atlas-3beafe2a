## What is Material 3?

Material 3 (also known as Material You) is Google's latest design system that introduces **personalization** as a core principle. Unlike previous Material Design versions, Material 3 creates unique color schemes derived from user preferences, wallpapers, and system settings.

<Info>
  Material 3 brings color personalization to Android apps, allowing your app's theme to automatically adapt to each user's device preferences.
</Info>

Key features of Material 3:
- **Dynamic color**: Colors generated from the user's wallpaper or preferences
- **Expressive surfaces**: Enhanced elevation and surface styling
- **Adaptive components**: UI elements that respond to context
- **Accessible by default**: Built-in accessibility features

## How Dynamic Color Works

Dynamic color is available on **Android 12 (API level 31) and higher**. The system extracts a color palette from the user's wallpaper and applies it consistently across all apps that support Material 3.

<Info>
  When users change their wallpaper, the system automatically generates a new color scheme. Apps using `expo-material3-theme` will receive these updated colors.
</Info>

### The Color Extraction Process

1. **Source extraction**: Android analyzes the user's wallpaper to identify prominent colors
2. **Palette generation**: The system creates a complete color palette using Material Design's color algorithms
3. **Theme distribution**: Apps can access these colors through the system API
4. **Automatic updates**: When wallpaper changes, themes update automatically

### Platform Support

Dynamic color support varies by platform:

| Platform | Dynamic Color Support | Fallback Behavior |
|----------|----------------------|-------------------|
| Android 12+ | ✅ Full support | N/A |
| Android 11 and below | ❌ Not supported | Uses custom theme from source color |
| iOS | ❌ Not supported | Uses custom theme from source color |

You can check for dynamic theme support in your code:

```tsx
import { isDynamicThemeSupported } from '@pchmn/expo-material3-theme';

if (isDynamicThemeSupported) {
  // Android 12+: System theme is available
} else {
  // Fallback to custom theme
}
```

See [index.ts:12](https://github.com/pchmn/expo-material3-theme/blob/main/src/index.ts#L12) for the implementation.
